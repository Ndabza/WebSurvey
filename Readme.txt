## ✨ About

capturing user input through a survey and displaying the statistics on a separate, dynamic page.

## 🚀 Features

*   **User-Friendly Survey Interface:** A clean and easy-to-use form for users to provide their input.
*   **Real-Time Statistics Dashboard:** An interactive page that instantly updates with new survey results.
*   **Data Capture and Storage:** Survey responses are securely captured and stored in sqlite database.
